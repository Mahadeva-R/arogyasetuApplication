## 📱 ArogyaSetu Application (Automation/Test Project)

This project is a Selenium-based automation framework built to test the functionality of the Aarogya Setu application (or similar web/mobile workflows).

It demonstrates:
- Automated browser testing using Selenium WebDriver
- ChromeDriver integration
- Test execution and validation scenarios

## 🛠 Tech Stack
- Java
- Selenium WebDriver
- ChromeDriver
- WebDriverManager
- TestNG
- log4j
- Maven build tool
